Sammlung aus 60+ Tools, kleineren und größeren Codeblöcken sowie vollständigen
Skripten, die im Rahmen von Arbeit mit Geodaten
entstanden sind. Über die Zeit werden hier Dateien zusammengetragen, um 
routinemäßig anfallende Arbeitsschritte zu beschleunigen. Die Skripte werden kontinuierlich überarbeitet und bauen teilweise aufeinander auf bzw. ergänzen sich. Die Python-Funktionen einiger Skripte sind aufeinander abgestimmt, sie können also als zusammenhängende Toolbox betrachtet werden. Im Ordner "Skripte" sind verschiedene Ordner für Python, R und separate Jupyter Notebooks angelegt. Ergänzende Jupyter Notebooks befinden sich ebenfalls bei den .py-Dateien unter "Python". 

Aktuell werden einige der Skripte zu vorhandenen Workflows zusammengefügt: Siehe dafür den Wasserrückhalt_Datenauswertungs_Workflow.md (Arbeitsschritte zur Bearbeitung von Projekten im Wassermengenmanagement mit entsprechenden Verweisen auf Skripte) oder auch die flurabstands_processing.R mit Funktionen zum Rasterisieren, Hochskalieren, Anpassen der räumlichen Ausdehnung und aktualisierten TIF-Generierung für Modellierungsergebnisse von Wasserständen. 

Der fachliche Fokus liegt auf Hydrologie und Hydrogeologie, Fernerkundung, Geoökologie, Geodatenverarbeitung und Machine Learning für Geodatenanalyse.  

Für eine ausführliche Beschreibung aller Skripte sowie ihrer Anwendungen und technischen Besonderheiten sei auf die OVERVIEW.md - Datei verwiesen.
